This project was developed in a team formed for the SWEN 261 class at RIT. Focusing on: designing a software system, using a team repository (SVN), and working in a team. This project took a semester to complete, starting with the requirements phase we had a three staged release throughout the semester. My role in the team was "Team Leader" which entailed keeping track of the teams progess, updating our progress tracker, and reviewing/submitting all items. This was a full stack construction using django/python for the backend with a mysql3 database, also utilizing bootstrap/html/css/js in the front end.For more information about this project/source code please fill out the contact me form in the footer of the home page.
